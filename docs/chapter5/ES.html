查看mapping

统计某个字段的数目

http://host:9200/_sql?sql=SELECT count(distinct(resource_id)) as num FROM index_*

查看索引数据

POST 

http://host:9200/index_*/_search?pretty

Body

{
    "size":1000
}

删除索引

curl -XDELETE '127.0.0.1:9200/index_*'
